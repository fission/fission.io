## fission canary list

List canary configs

### Synopsis

List all canary configs in a namespace if specified, else, list canary configs across all namespaces

```
fission canary list [flags]
```

### Options

```
  -A, --all-namespaces   -A |:|: Fetch resources from all namespaces
  -h, --help             help for list
```
