## fission watch list

List kube watchers

### Synopsis

List all kube watchers in a namespace if specified, else, list kube watchers across all namespaces

```
fission watch list [flags]
```

### Options

```
  -A, --all-namespaces   -A |:|: Fetch resources from all namespaces
  -h, --help             help for list
```
